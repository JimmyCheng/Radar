About

CodeRadar is for enterprise private repository search. Normally it is deployed in company's intranet and available for developpers to locate the code.

Why Private Repository Tool

This Tool could be quite useful for developers to debug the code. Imagine that an enterprise product has several release and million lines of code. How to locate the code? With this tool, not only the source code, but the configuration file, resource file can be located.

This website has demonstrated the Tomcat Repository, with different versions.

How this tool works

CodeRadar is powered by Solr/Lucent. It can supports full text search.
